Intrusion Detection – Comparative ML Study

Machine Learning Research Project

Overview

A comprehensive cybersecurity research project addressing the critical need for intelligent network threat detection. Developed and evaluated a machine learning pipeline comparing 7 classification algorithms (Random Forest, DNN, SVM, Naive Bayes, Decision Tree, Logistic Regression, and KNN) to identify malicious network activities and potential security breaches using the NSL-KDD and IoT-specific datasets.

Key Security Contributions
  • 90% detection accuracy using Random Forest and DNN models for identifying DoS, Probe, R2L, and U2R attacks
  • Evaluated model performance against multi-class attack scenarios including modern IoT-based threats
  • Optimized false positive rates to minimize security alert fatigue in production environments
  • Demonstrated practical applications for real-time network monitoring and threat intelligence

Built with:

Python Scikit-learn Pandas NumPy Matplotlib TensorFlow/Keras

HeartCare AI – Cardiac Arrhythmia Detection Platform

Data Engineering & Healthcare AI Project

Overview

Designed a comprehensive proposal for an AI-powered heartbeat detection platform using Deep Neural Networks. Developed an automated Python data pipeline processing 650k+ ECG healthcare records, transforming raw cardiac data into actionable insights for real-time arrhythmia detection. The project integrates data engineering, machine learning, and healthcare analytics to improve early intervention and patient outcomes.

Key Achievements & Impact
  • Processed 650,000+ ECG records with automated data cleaning and validation pipelines
  • Designed wearable device integration concept for continuous cardiac surveillance
  • Improved data accuracy through systematic preprocessing and quality checks
  • Created professional documentation including technical reports and presentation slides
  • Reduced manual processing time by implementing automation workflows
